# 🛒 EC App

A simple e-commerce web application built with React, TypeScript, and Supabase.

## Features

- Product listing
- Product detail page
- Shopping cart
- Guest checkout
- Form validation
- Order data stored in Supabase
- Clear cart after checkout

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase

## Setup

```bash
git clone <repo-url>
cd ec-app
npm install
npm run dev
