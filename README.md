# EC Cart App

A simple EC-style cart application built with React and TypeScript.

## Features

- Product list display
- Add to cart
- Update quantity
- Total price calculation

## Tech Stack

- React
- TypeScript
- Vite

## Purpose

This project was created to understand the difference between a Todo app and a more complex state management scenario involving multiple related data and calculations.
